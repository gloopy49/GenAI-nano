                 GenAI Enpowered Property Matching System

Project Overview
The project implements an intelligent property matching system leveraging Large Language Models (LLMs) and embedding-based similarity search to recommend home buyers with ideal properties. 

Key Features
1.Synthetic property listing generation using LLMs
2.Semantic search using embedding-based similarity
3.Personalized property recommendations
4.AI-powered listing refinement based on user preferences

Technologies Used
Python 3.12+: Core programming language
OpenAI API: For LLM-based text generation and embeddings
LangChain: Framework for LLM application development
ChromaDB: Vector database for similarity search
Pandas: Data manipulation and analysis
